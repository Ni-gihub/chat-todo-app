# Chat×Todoアプリ
## 概要
チャット感覚でタスク管理ができるWebアプリ
- チーム内でのタスク追加・確認がチャット形式で可能
- 自然言語コマンドでタスク名・期限・担当者を指定可能
- 小規模チームや学生のプロジェクト向けに開発

## 対象ユーザー
- 小規模チーム（学生や趣味のプロジェクトチーム）
- 日常的にタスク管理を行うが、シンプルに操作したいユーザー

## 技術スタック
- フロントエンド: HTML, CSS, JavaScript
- バックエンド: Firebase Authentication（メール/パスワード認証、Googleログイン対応）、Firestore
- デプロイ：

## 機能一覧
- ユーザー登録・ログイン
  - メール/パスワード認証
  - Googleアカウントログイン
- タスク管理
  - タスクの追加、編集、削除
  - 担当者指定
  - 期限設定
- チャット機能
  - タスクをチャット形式で表示
  - リアルタイム更新（他のユーザーにも即時反映）
- ルーム・チャンネル管理
  - ルームごとにチャンネル作成
  - チームやプロジェクト単位でのタスク管理
